Install nodeJS : https://nodejs.org/en
Install MongoDB : https://www.mongodb.com/try/download/community-kubernetes-operator

1. create a folder/frontend_backend
2. open vs code
3. open terminal 
4. npm init -y

5. create index.js in the main folder
6. install some dependencies/
npm install express 
npm install body-parser
npm install mongoose 
npm install nodemon

7. write code in index.js [explain]

8. Now write the HTMl file for Signup registration form and signup thankyou page - create a Public folder in the main directory
9. create index.html in public folder 
10 create style.css in public folder
11. signup_success.html

12. run nodemon index.js
13. start mongoDB
14. start connection on mongodb

15. run localhost:3000 in any browser
16. fill the form - submit - now check mongoDB - new DB has been created and data has been submitted
